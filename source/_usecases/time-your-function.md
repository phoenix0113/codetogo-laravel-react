```javascript
function foo(){
    // do stuff
}

let start = performance.now();
foo();
let end = performance.now();

console.log(`Call to ${foo.name} took ${(end - start)} milliseconds.`);
```

<pre class="output">Call to foo took x milliseconds</pre>
